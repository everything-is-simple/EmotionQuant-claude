# SpiralRoadmap (New System)

本目录用于新系统实现路线图，独立于 `Governance/Capability/` 的能力包契约。

## 定位

- `Governance/Capability/`：定义能力包（CP）契约与依赖
- `Governance/SpiralRoadmap/`：定义新系统按圈推进的实现路线

## 闭环规则

每个 Spiral 必须具备以下五项证据才能收口：

- `run`
- `test`
- `artifact`
- `review`
- `sync`

## 目录结构

- `SPIRAL-IMPLEMENTATION-OVERVIEW.md`：螺旋总览与当前圈
- `spirals/SPIRAL-SXX-template.md`：单圈模板
- `spirals/SPIRAL-S00-bootstrap.md`：首圈启动样例
