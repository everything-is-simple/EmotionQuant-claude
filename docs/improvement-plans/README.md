# EmotionQuant 改进行动计划目录

## 1. 目录用途

本目录是系统“改进行动计划”的唯一固定入口。

目标: 解决计划文件分散、难定位的问题。

## 2. 使用规则

1. 当前正式文档保留 2 份：
   - 执行主计划：`docs/improvement-plans/eq-improvement-plan-core-frozen.md`
   - 外挂系统权威设计：`docs/improvement-plans/enhancement-selection-analysis_claude-opus-max_20260210.md`
2. 临时草稿仅存放在 `docs/improvement-plans/drafts/`，只能参考，不可作为权威引用。
3. 新一轮优化必须回写正式文档并补充变更记录，不新增同类平行“正式主计划”。

## 3. 阅读顺序

1. 先看主计划: `docs/improvement-plans/eq-improvement-plan-core-frozen.md`
2. 再看外挂权威设计: `docs/improvement-plans/enhancement-selection-analysis_claude-opus-max_20260210.md`
3. 需要溯源时再看 `docs/improvement-plans/drafts/` 与 `.reports/` 历史材料。
